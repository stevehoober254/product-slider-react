# REACT PRODUCT SLIDER COMPONENT

A simple responsive Product slider with autoplay scroll.

## Built with

* Create React App
* React Swiper
* React Bootstrap

### How to run

* Clone Repo

````bash
  git clone https://github.com/stevehoober254/product-slider-react.git
  ````

* Run

````bash
  npm install
  ````

* Run

````bash
  npm start
  ````
